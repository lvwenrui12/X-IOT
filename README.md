# X-IOT 工业互联网 系统

## 初步计划
 先实现数据的推模式，也就是 平台接收终端设备主动上报的信息。
 组态采用web模式。

---
## 技术栈：

### 前端
   * angular8
   * Ant Design
   * 组态 threejs
### 后端
   * Springboot，
   * Springcloud，
   * 中间件：kafka
   * GRPC
   * 数据库:mongo
   * orm框架: Springdata,Druid(多数据源配置)
